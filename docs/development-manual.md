# Development manual  

## Branch naming  
### Structure  
(prefix)/(short-name)/(year-month) without ( )  
### Allowed characters  
Whole branch name consists of lowercase A-Z, 0-9, and "/" with space character replaced by "-"  
### Prefixes
new - A new feature in code  
doc - A new aspect of documentation  
dir - Change related to project hierarchy and directories  
ref - Refactor of the code, does not affect functionality  
opt - Performance optimization, does not affect code functionality   
fix - Fix of the known bug  
exp - Miscellaneous/experimental feature
